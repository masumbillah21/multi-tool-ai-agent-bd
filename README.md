# 🇧🇩 Multi-Tool AI Agent v5 (FastAPI + Streamlit)

## Architecture
Frontend: Streamlit  
Backend: FastAPI (Groq + Tavily Hybrid Synthesis)

## Run with Docker

1️⃣ Copy environment file:
cp .env.example .env

2️⃣ Add your API keys inside `.env`

3️⃣ Run:
docker compose up --build

Frontend:
http://localhost:8501

Backend Docs:
http://localhost:8000/docs
